# ⌨ PhotoGIMP Keyboard Shortcuts

PhotoGIMP remaps GIMP's keyboard shortcuts to match [Adobe Photoshop's defaults (Windows)](https://helpx.adobe.com/photoshop/using/default-keyboard-shortcuts.html). On macOS, read <kbd>Ctrl</kbd> as <kbd>Cmd</kbd>.

Shortcuts appear next to the menu items inside GIMP, just like in Photoshop. You can change any of them via **Edit → Keyboard Shortcuts**.

## Notes on differences

- **Ctrl+S saves a `.xcf` project file** (like a `.psd`). Use **Ctrl+Alt+Shift+W** (Export As) for PNG/JPEG, matching Photoshop's Export As.
- **Ctrl+Shift+I** inverts the *selection*; **Ctrl+I** inverts *colors* — same split as Photoshop.
- Photoshop features without a GIMP equivalent (Reselect, Ungroup Layers, Clipping Mask, Content-Aware Scale) have no mapping.
- GIMP features that keep their GIMP defaults are untouched unless they collided with a Photoshop shortcut.
